# Fibhess Game (React + TypeScript + Vite)

This project implements the board game described in `requirements.md`. To start development:

1. **Install dependencies**: run `npm install` in the `fibhess` directory (uuid and react-dnd have been added).
2. **Start dev server**: `npm run dev` will launch Vite and open the app in the browser.

The application has two phases:

- **Setup** – red and blue players take turns dragging pieces from their player box onto the 10x10 board.
- **Play** – after all pieces are placed, players move pieces one square per turn; attacking occupied squares resolves strength sums (piece + surrounding 8 neighbours) as per rules.

Game state is managed via React context and a reducer (`src/GameContext.tsx`); drag-and-drop is handled with `react-dnd`.
